# Цель проекта:
***"AI in actions"***
Целью проекта является инфориация для создание платформы; разработки и обучения, развёртывания и запуск AI агентов. 

## Краткое описание
Проект представляет собой информационную систему для создания разносторонних мультиагентных систем. 
Которая позволяет:
* проектировать специализированных AI‑агентов с уникальными ролями;
* организовывать взаимодействие между агентами, службами и сервисами;
* обучать агентов не только для решения повседневных задач, но и  для управления бизнес процессами;
* мониторить и принимать решения на Финансовых, Фондовых и Криптовалютных рынках;
* анализировать производительность и поведение других AI агентов.

**Ключевые особенности:**
* модульная архитектура агентов;
* поддержка разных LLM (языковых моделей);
* инструменты для совместной работы агентов;
* API для интеграции с внешними сервисами;
* веб‑интерфейс для мониторинга;
* и тд.
