# Цель проекта:
***"AI Agents in actions"***
Целью данного проекта является создание платформы для разработки, обучения и развёртывания AI агентов ИИ. 

## Краткое описание
Проект представляет собой информационную систему для создания мультиагентных систем. 
Которая позволяет:
* проектировать специализированных ИИ‑агентов с уникальными ролями;
* организовывать взаимодействие между агентами;
* обучать агентов на пользовательских данных;
* мониторить производительность и поведение агентов.

**Ключевые возможности:**
* модульная архитектура агентов;
* поддержка разных LLM (языковых моделей);
* инструменты для совместной работы агентов;
* API для интеграции с внешними сервисами;
* веб‑интерфейс для мониторинга.
